실행방법
1.	make clean을 이용해 object파일을 지워줍니다.
2.	make를 통해서 컴파일을 하여 실행파일인 snake를 생성합니다.
3.	snake를 실행하게 되면 아래 이미지(level_1)와 같이 게임이 시작됩니다.
4.	혹은 ./run.sh명령어로도 실행할 수 있습니다. (bin/bash를 실행해서 수행)
